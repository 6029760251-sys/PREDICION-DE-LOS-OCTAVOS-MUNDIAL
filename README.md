Este proyecto implementa modelos de Machine Learning para predecir los resultados del Mundial FIFA 2026 utilizando información histórica de las Copas del Mundo.

## Características
- Entrenamiento de modelos de Regresión Logística y Random Forest.
- Evaluación mediante la métrica Accuracy.
- Simulación completa del torneo:
  - Octavos de final
  - Cuartos de final
  - Semifinales
  - Partido por el tercer puesto
  - Final
- Predicción del campeón, subcampeón y tercer lugar.
- Comparación del rendimiento entre ambos modelos.

## Tecnologías
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook
